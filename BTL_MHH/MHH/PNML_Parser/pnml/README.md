# PNML Models

This directory contains PNML files used for testing and demonstrating the capabilities of the PNML parser implemented in this project.
The models in directory:

- `simple`
- `medium`
- `large`

are all sourced from this [GitHub repo](https://github.com/dalzilio/mcc/tree/master).

The models in directory:

- `examples`

is sourced from this [PNML Organization](https://www.pnml.org/version-2009/version-2009.php).

Feel free to explore and utilize these PNML files for your own testing and development purposes!
